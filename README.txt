Due to the fact that most parts of the module wereredone there are many changes in the language files.Chapter 2: The installation1. Copy the xoopschat-folder to modules-folder of your Xoops.2. Go to Admin CP and install via modules-administration.3. Make your 'Settings' for the module.4. Go to the groups-administration and grant the rights to the module.5. If necessary go to the blocks-administration and make the shoutbox-block visible.Chapter 3: Notes1. register_globals must be ON!2. At the moment the immediate logoff from the chat can only be achieved bypressing the "Leave Chat"-Button. If you do not leave the chat in this waythe user will be still counted as online on the chat.3. Anonymous users are NOT allowed to use the chat.4. See "credits.txt" for information about who created this module.Chapter 4: A few things to do1. Chat working with register_globals OFF2. Allow anonymous users to use the chat.3. Automatic logoff from chat.4. Enhancing chat performance.5. Templates (if needed, but the adaption of the code is very simple).6.
